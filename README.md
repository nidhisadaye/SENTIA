# SENTIA 🦯🤖

AI-powered voice assistant for visually impaired users, providing real-time navigation, object detection, OCR text reading, emergency SOS, and conversational assistance.

## Features

- 🎤 Voice Assistant
- 🗺️ Real-Time Navigation
- 📍 Current Location Awareness
- 👁️ Object Detection
- 📖 OCR Text Reading
- 🚨 Emergency SOS
- 🤖 Conversational AI
- 🌐 Multilingual Support (English, Hindi, Marathi)

## Voice Commands

- Where am I?
- Navigate to [destination]
- Start Walk With Me
- Stop Navigation
- Help
- SOS

## Tech Stack

- React Native
- Expo SDK 54
- TypeScript
- FastAPI
- Docker
- OpenAI / Groq
- Expo Speech Recognition
- Expo Location
- Expo Camera

## Project Goal

SENTIA is designed to help visually impaired individuals navigate, understand their surroundings, and interact with technology through a voice-first AI experience.
